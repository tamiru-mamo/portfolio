E-Commerce Website for Local Retailer

Modernizing local business with an intuitive online store

Overview

This project aimed to bring a local retail shop online. I designed a user-friendly shopping experience with an emphasis on clean layout, smooth navigation, and an easy-to-use admin dashboard for non-technical staff.

Tools & Technologies

  • HTML5 & CSS3
  • JavaScript for cart logic and filtering
  • Responsive Design (Flexbox & Grid)
  • Mock JSON or basic CMS integration (optional)

Website Screens

Key Features

  • Product listings with image, description, and price
  • Add-to-cart and checkout flow
  • Search and category filter options
  • Simple admin dashboard for managing inventory

Challenges & What I Learned

Building the cart system with basic JavaScript helped strengthen my DOM skills. I also focused on making the admin panel intuitive for users with little or no tech background, which required simplifying the UI and testing with feedback.
