Blue Note Bistro Website

A bold, music-driven experience for a jazz café

Overview

Blue Note Bistro is a vibrant jazz café that needed a bold online presence to match their energetic live performances. I designed a website that immerses users in the atmosphere of the venue while allowing them to discover events and make reservations easily.

Tools & Technologies

  • HTML5 & CSS3
  • JavaScript for dynamic menus
  • Responsive Design (Mobile First)
  • Embedded event calendar

Website Screens

Key Features

  • Event discovery with embedded jazz calendar
  • Stylish dark-mode inspired color palette
  • Interactive reservation form
  • Responsive layout for mobile, tablet, and desktop

Challenges & What I Learned

I focused on creating a high-impact visual identity that still worked responsively. One challenge was managing dynamic content like event listings on mobile. I also improved my skills in designing with accessibility in mind — including contrast, readability, and keyboard-friendly forms.
